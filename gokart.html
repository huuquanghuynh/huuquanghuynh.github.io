Refining Autonomous Go-Karts: Advanced Control, Path Planning, and Localization Prototyping

Key Takeaways

  • Achieved a 100x improvement in braking response time for a racing go-kart.
  • Enhanced localization accuracy using advanced sensor fusion techniques.
  • Demonstrated real-world applications of control systems and state estimation.

Objective

  • Design an improved braking system for racing scenarios to ensure rapid response and safety.
  • Enhance localization precision for a fully autonomous go-kart using multi-sensor data integration.

Approach

  • Braking System:
    • Designed a hydraulic pump-based system for faster brake fluid compression.
    • Integrated a PID algorithm to match desired and actual brake pressure.
    • Utilized PWM signals for precise hydraulic pump control.
  • Localization:
    • Implemented an Extended Kalman Filter (EKF) to fuse GPS and IMU data.
    • Developed the EKF in Python to improve state estimation accuracy.

Results

  • Braking System:
    • Reduced brake response time to 20-30 milliseconds, a significant improvement from 1-2 seconds.
    • Enhanced safety and competitiveness in racing scenarios.
  • Localization:
    • Improved localization accuracy through GPS and IMU integration.
    • Ensured precise and reliable navigation for autonomous go-kart operation.
